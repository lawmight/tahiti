# NEXT_STEPS

## Priorité haute

1. **Filtres croisés sur le RNTE**
   - ajouter des filtres par commune, NAF, statut actif/radié et classe d’effectifs ;
   - permettre de passer d’une vue « top 10 » à une exploration plus libre.

2. **Analyses avancées sur le tourisme**
   - comparaison interannuelle glissante ;
   - variation YoY / saisonnalité normalisée ;
   - focus sur les croisiéristes et les nuitées.

3. **Exploration cartographique**
   - carte des communes pour le RNTE ;
   - représentation territoriale des concentrations d’activité.

## Priorité moyenne

4. **Croisements entre jeux de données**
   - comparer la dynamique du trafic aérien et celle du tourisme ;
   - relier les principaux partenaires commerciaux avec les flux touristiques.

5. **Enrichissement des libellés NAF et juridiques**
   - compléter les correspondances des formes juridiques les plus rares ;
   - améliorer les libellés affichés pour les secteurs NAF spécialisés.

6. **Détection d’anomalies / ruptures**
   - signaler les mois atypiques dans les séries longues ;
   - mettre en évidence les ruptures structurelles au-delà du choc COVID.

## Priorité basse

7. **Exports utilisateurs**
   - export PNG / CSV des graphiques affichés ;
   - partage d’une vue configurée via l’URL.

8. **Comparateurs multi-onglets**
   - panneau de synthèse transverse ;
   - sélection de plusieurs indicateurs clés venant de modules différents.

9. **Améliorations d’ergonomie**
   - recherche dans les onglets ;
   - légendes interactives ;
   - mémorisation du dernier onglet consulté.
